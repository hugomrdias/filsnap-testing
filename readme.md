# Filsnap Testing Monorepo

## Install

```bash
# install all dependencies in the mono-repo
pnpm install
# setup git hooks
npx simple-git-hooks
```

## Packages

### [tools](./packages/tools)

This is a collection of tools for testing metamask, metamask flask and metamask snaps.
